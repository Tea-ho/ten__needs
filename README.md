# ten__needs
tenNeeds(project3)

#### 1. 주제
테니스 웹(미니게임, 채팅, 근처 체육시설 정보 제공 서비스)

#### 2. 주제 선정 이유
1) MVC 패턴 학습
2) CRUD 기능 적용
3) sockat을 이용한 2인 전용 게임 서비스 추가
4) 공공데이터 및 kakao API 사용
5) SQL 활용(DB관리 및 통계처리)
